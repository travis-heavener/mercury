Arch Installation

README.md has full instructions on how to build Mercury (see For Developers).
To build Mercury on Arch Linux:

  1. Clone the official Mercury repository.
  2. Build dependencies via make libs -j or make libs -j LINUX_ONLY=1 (to omit Windows libraries).
  3. Run make linux.
  4. Mercury can now be run from bin/mercury.

† Linux copies of Mercury require glibc to be present and currently support distros with APT or Pacman.
If you encounter any compatibility issues, please open an Issue.
